# Social Media Clone

Facebook-inspired social app with posts, likes, comments, follow system. React + Node.js + Socket.io.

> React 18 · Node.js · Express · Socket.io · SQLite · JWT · React Router

## Features
- Register / Login with JWT
- Create and delete posts
- Like / unlike posts
- Comment on posts
- Real-time new posts via Socket.io
- Follow / unfollow users
- User profiles with stats

## Quick start

```bash
# Backend
cd backend
copy .env.example .env
npm install && npm run dev

# Frontend
cd frontend
npm install && npm start
```

Frontend: http://localhost:3000
